# 🚗 Rush Hour Game Solver

Proyecto desarrollado en el ámbito de la asignatura de programación, basado en la implementación del clásico juego Rush Hour.

## 🚀 Descripción

El proyecto consiste en el desarrollo de una aplicación capaz de representar y resolver el juego Rush Hour, donde el objetivo es desbloquear el vehículo principal moviendo el resto de vehículos en un tablero.

Se ha trabajado en la lógica del juego, representación del estado y gestión de movimientos.

## 🧠 Funcionalidades

- Representación del tablero del juego
- Gestión de vehículos y posiciones
- Validación de movimientos
- Resolución del juego mediante algoritmos
- Interfaz de ejecución por consola

## 🛠️ Tecnologías

- Lenguaje de programación (C / Java / Python) ← (pon el tuyo)
- Estructuras de datos
- Algoritmos de búsqueda

## 🧩 Lógica del juego

El sistema modela el tablero como una estructura de datos donde cada vehículo ocupa una posición específica.

Se permite el movimiento de vehículos en función de reglas definidas, buscando liberar el camino del coche principal hasta la salida.
